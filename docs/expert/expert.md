# 专家等级要求

专家等级如何分，我个人水平也达不到，还需要继续摸索。

但是个人感觉要在具备服务高性能高可用的前提下，具备分布式服务的能力。

分布式结构就是将一个完整的系统，按照业务功能，拆分成一个个独立的子系统，在分布式结构中，每个子系统被称为服务。而且这些服务能独立运行在web容器中。比如SOA（面向服务的架构）和最近大热的微服务，其中最常用的框架有：
- Spring Cloud
- Dubbo
